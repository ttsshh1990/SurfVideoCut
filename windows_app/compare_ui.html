Local Compare Workspace

Drop a DOCX and a saved HTML page. Get back a browser-rendered PDF with embedded difference comments.

This interface runs the local comparison pipeline, renders the webpage through Chromium, and annotates only the meaningful content differences against the Word source.

Exact browser rendering The PDF is created from the saved HTML page itself, not a text-only reconstruction.
Drag-and-drop flow Drop files anywhere on their panels or click to pick them manually.
Review-ready output The generated PDF can be downloaded immediately and previewed in the browser.

Inputs

Use one Word `.docx` file and one saved webpage `.html` or `.htm` file. The compare job runs locally on this machine.

Waiting for both files.
The server stores each run in a local `ui_runs/` folder so you can reopen the PDF and JSON summary after generation.

No Compare Run Yet

Once both files are loaded, the generated PDF and compare summary will appear here.
